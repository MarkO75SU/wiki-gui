1. Getting Started

  • Q: What is WikiGUI?A: WikiGUI is a web-based tool that assists users in building complex search queries for Wikipedia by providing an intuitive interface for various search parameters.
  • Q: Is WikiGUI free to use?A: WikiGUI is provided under an open license. Basic use is free. Please refer to the license files for details.
  • Q: Do I need to install anything to use WikiGUI?A: No, WikiGUI is a web application. You can use it directly from your web browser without any installation.

  • Q: How does the "All these words" field work?A: Entering terms in the "All these words" field means all of them must be present in the search results.
  • Q: How does the category dropdown work?A: The "Category" dropdown allows you to select a predefined Wikipedia category to pre-fill the search parameter.

3. Wikipedia vs. AI-Search

  • Q: Why should I use Wikipedia when I can ask AI (like ChatGPT)?A: AI models can "hallucinate" (invent facts). Wikipedia provides transparency through citations. WikiGUI also allows for targeted metadata searches (file types, dates) which AI cannot do.
  • Q: What is the advantage of WikiGUI over a simple AI answer?A: WikiGUI treats Wikipedia like a database. You can search for specific original sources (like PDFs), while AI usually only provides a summary.

4. Troubleshooting & Support

  • Q: How do I report a bug or ask a question?A: Use the "Report an Issue" link in the footer. This will take you to our GitHub Issues page.
