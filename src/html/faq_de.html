1. Erste Schritte

  • F: Was ist WikiGUI?A: WikiGUI ist ein webbasiertes Werkzeug, das Benutzer dabei unterstützt, komplexe Suchanfragen für Wikipedia zu erstellen, indem es eine intuitive Oberfläche für verschiedene Suchparameter bietet.
  • F: Ist die Nutzung von WikiGUI kostenlos?A: WikiGUI wird unter einer offenen Lizenz zur Verfügung gestellt. Die grundlegende Nutzung ist kostenlos. Bitte beachten Sie die Lizenz-Dateien für Details.
  • F: Muss ich etwas installieren, um WikiGUI zu nutzen?A: Nein, WikiGUI ist eine Webanwendung. Sie können es direkt in Ihrem Webbrowser ohne Installation nutzen.

2. Features & Funktionalität

  • F: Wie funktioniert das Feld "Alle diese Wörter"?A: Die Eingabe von Begriffen in das Feld "Alle diese Wörter" bedeutet, dass alle von ihnen in den Suchergebnissen vorhanden sein müssen.
  • F: Wie funktioniert das neue Kategorie-Dropdown?A: Das "Kategorie"-Dropdown ermöglicht es Ihnen, eine vordefinierte Wikipedia-Kategorie auszuwählen, um den Suchparameter "In Kategorie" schnell auszufüllen.

3. Wikipedia vs. KI-Suche

  • F: Warum sollte ich Wikipedia nutzen, wenn ich KI (wie ChatGPT) fragen kann?A: KI-Modelle können "halluzinieren" (Fakten erfinden). Wikipedia bietet Transparenz durch Quellenangaben. WikiGUI ermöglicht zudem die Suche nach Metadaten (Dateitypen), was KI nicht kann.
  • F: Was ist der Vorteil von WikiGUI gegenüber einer einfachen KI-Antwort?A: WikiGUI macht Wikipedia zu einer Datenbank. Sie können gezielt nach Originalquellen (z.B. PDFs) suchen, während KI meist nur zusammenfasst.

4. Fehlerbehebung & Support

  • F: Wie kann ich einen Fehler melden oder eine Frage stellen?A: Verwenden Sie den Link "Problem melden" in der Fußzeile. Dieser führt Sie zu unserer GitHub-Issues-Seite.
